Challenge

The goal of this lab was to create Fancy CSS. Basically, this means, with our coding knowledge, Wes wants us to embellish and add more CSS elements. This will help us stylize our pages. I used the text-shadow, box-shadow, gradient,, filters, rounded corners, object-fit, and transform elements.

Problems

The biggest problem I had when creating it was making sure it looked nice but not guady with CSS. Adding text shadows and a lot of styling elements can often become overdramatic.
