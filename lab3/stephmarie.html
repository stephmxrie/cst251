Lab 3: File Structures

Objective:

The lab focused on creating file structures. File structures consists of directories, folders, and files. With learning the difference between these three we also learned how to understand the layout template and create and expand the overall layout.

  • Directories: a placeholder for all folders
  • Folder: contains other folders and files, mostly subfolders
  • File: contains data (.html, .jpeg, etc)

Challenges:

The lab was overall easy to understand and comprehend. Most of the labs usually correspond with the lecture, therefore the difficiulty is minimal. Along with learning file structures we also learned how to add pictures. Adding pictures was an easy task, however, the difficulty is to make sure how to write file names down verbatim and make sure you have the right file type. (.jpeg, .png)
